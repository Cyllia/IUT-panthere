# IUT-panthere
Agence La Panthère

Web design & stratégie digitale à Lyon

#Description
Ce projet est le site vitrine de l’Agence La Panthère, une agence web spécialisée dans le web design, la stratégie digitale et la création d’illustrations.
Le site met en avant les services de l’agence, sa vision, ainsi qu’un portfolio de réalisations, avec une attention particulière portée au référencement naturel (SEO) et à l’expérience utilisateur.

#Objectifs du site
- Présenter l’agence et son identité visuelle
- Mettre en avant les services proposés
- Valoriser les réalisations via un portfolio
- Inciter les visiteurs à prendre contact
- Optimiser la visibilité sur les moteurs de recherche

#Fonctionnalités principales
- Page d’accueil responsive
- Navigation simple (Accueil / Contact)
- Section services (Web design, Stratégie, Illustrations)
- Section présentation de l’agence
- Portfolio avec aperçu des projets
- Bouton d’appel à l’action (CTA) vers la page Contact
- Intégration des réseaux sociaux
- Optimisation SEO (balises meta, attributs alt, structure sémantique)

#SEO & Accessibilité
- Balises meta optimisées (description, keywords, viewport)
- Titres hiérarchisés (h1, h2, h3)
- Attributs alt descriptifs sur les images
- Liens accessibles avec aria-label
- Contenu orienté mots-clés locaux (Lyon, agence web, web design)

#Installation & utilisation
Télécharger ou cloner le projet
Ouvrir index.html dans un navigateur web
Aucune configuration serveur requise (site statique)

#Localisation
Agence basée à Lyon, spécialisée dans l’accompagnement des entreprises locales.
